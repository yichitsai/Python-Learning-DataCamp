# Customization

### Add title
```
title()

plt.title()
```
### Add label
```
xlabel()
ylabel()

plt.xlabel()
```

### Ticks
```
xticks([object1, object2])
yticks([object1, object2])
#replace object1 to object2

plt.xticks([,])
plt.yticks([,])
```

### Sizes
```
plt.scatter(data1, data2, s = ) 

# s=size (can be match to the population,amount, or so on.)
```

### Colors
```
plt.scatter(data1, data2, s = , c=col, alpha= )
# c=color
# alpha=0~1 透明度
```

### Add grid
```
plt.grid(True)
```
